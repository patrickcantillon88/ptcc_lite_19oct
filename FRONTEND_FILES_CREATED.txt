================================================================================
                              QUICK START
================================================================================

1. Copy the 6 new component files to your frontend:
   - src/services/agentsApi.ts
   - src/components/AgentsPage.tsx (+ .css)
   - src/components/AgentAnalysis.tsx (+ .css)
   - src/components/AgentBriefing.tsx (+ .css)

2. Import in your app:
   import AgentsPage from './components/AgentsPage';

3. Use it:
   <AgentsPage />

4. Start backend:
   cd backend && python -m backend.main

5. Test in browser at http://localhost:5174
